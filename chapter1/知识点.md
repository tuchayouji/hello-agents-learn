# 第一章 初识智能体

> **一句话总结**：智能体 = 感知 + 自主决策 + 行动，核心是 Thought-Action-Observation 循环

---

## ⭐ 核心定义

**智能体**：通过传感器感知环境 → **自主**决策 → 通过执行器行动的实体

```
┌─────────────────────────────────────────────┐
│           智能体四要素                        │
├──────────┬──────────┬──────────┬─────────────┤
│  环境    │  传感器  │  执行器  │  自主性     │
│ (外部世界)│ (获取信息)│ (改变环境)│ (独立决策)  │
└──────────┴──────────┴──────────┴─────────────┘
```

---

## ⭐ 智能体演进路线

```
简单反射 → 基于模型 → 基于目标 → 基于效用 → 学习型
  │           │           │           │          │
恒温器     自动驾驶    GPS导航    多目标权衡   AlphaGo
```

**关键转折**：从"预设规则"到"自主学习"

---

## ⭐ LLM 驱动的新范式

### 智能体循环（Agent Loop）

```
┌──────────────────────────────────────────┐
│  感知 → 思考(规划+工具选择) → 行动 → 观察 │
│    ↑                                    │
│    └────────────── 循环 ←───────────────┘
└──────────────────────────────────────────┘
```

### Thought-Action-Observation 范式

```python
Thought: 我需要查询天气
Action: get_weather("北京")
Observation: 北京当前天气:晴，气温26摄氏度
```

---

## ⭐ 三种分类维度

| 维度 | 类型 | 特点 |
|------|------|------|
| **决策架构** | 反射/模型/目标/效用/学习型 | 从简单到复杂 |
| **时间反应性** | 反应式/规划式/混合式 | 速度 vs 质量 |
| **知识表示** | 符号主义/亚符号主义/神经符号主义 | 规则 vs 数据 |

---

## ⭐ Workflow vs Agent

| 对比项 | Workflow | Agent |
|--------|----------|-------|
| **本质** | 按部就班执行指令 | 自主推理达成目标 |
| **灵活性** | 低（预设流程） | 高（动态决策） |
| **适用场景** | 固定流程 | 开放问题 |

## 📝 PEAS 模型

- **P**erformance：性能度量
- **E**nvironment：环境
- **A**ctuators：执行器
- **S**ensors：传感器
