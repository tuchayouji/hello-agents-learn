# 第二章 智能体发展史

> **一句话总结**：从符号主义到神经网络，从规则驱动到数据驱动，最终融合为现代 LLM 智能体

## ⭐ 符号主义时代 (1950s-1980s)

**核心思想**：智能 = 符号操作 + 逻辑推理

### 物理符号系统假说 (PSSH)
- 提出者：Newell & Simon (1976)
- **充分性**：物理符号系统 → 通用智能
- **必要性**：通用智能 → 物理符号系统

### 专家系统架构
```
┌─────────────┐    ┌─────────────┐
│   知识库    │◄───│   推理机    │───► 用户
│ (IF-THEN规则)│    │ (正向/反向链)│
└─────────────┘    └─────────────┘
```

### ⚠️ 符号主义四大局限

| 局限 | 说明 |
|------|------|
| **知识获取瓶颈** | 手工编码成本高、难扩展 |
| **常识问题** | 无法穷举世界知识 |
| **框架问题** | 动态世界状态追踪困难 |
| **系统脆弱性** | 规则外情况直接崩溃 |

---

## ⭐ 心智社会理论 (明斯基, 1986)

**核心观点**：
- 智能源于大量**简单智能体的协作**
- 复杂行为从简单交互中**涌现**
- **无中央控制器**，去中心化

**对多智能体系统的启发**：去中心化控制、涌现式计算、智能体社会性

---

## ⭐ 学习范式演进

### 联结主义 (1980s-)
- 神经网络 + 反向传播
- 知识分布式存储在**连接权重**中
- 解决了泛化能力问题

### 强化学习
```
状态(S) → 行动(A) → 奖励(R) → 更新策略 → 循环
```
- 核心：试错学习，最大化累积奖励
- 代表：AlphaGo

### 预训练范式
1. **预训练**：海量数据学习通用知识
2. **微调**：少量数据适配特定任务
3. **涌现能力**：上下文学习、思维链推理
